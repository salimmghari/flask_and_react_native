## Launch:

> - `cd backend/`
> - `pip install -r requirements.txt && python main.py`

> - `cd frontend/`
> - `npm install && npm start`
> - `npm run web`
> - `npm run windows`
> - `npm run macos`
> - `npm run android`
> - `npm run ios`
