### Launch:

#### Backend:

> - `cd backend/`
> - `pip install -r requirements.txt`
> - `python main.py`

#### Frontend:

> - `cd frontend/`
> - `npm install`
> - `npm run web`
> - `npm run windows`
> - `npm run macos`
> - `npm run android`
> - `npm run ios`
